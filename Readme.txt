---------------------------------------------------------------------------------------
HEX tools for Secp256k1 | Bitcoin (version 2.6)
---------------------------------------------------------------------------------------

Многофункциональный веб-инструмент для работы с HEX-числами и генерации Bitcoin адресов.

Calculator (HEX):
- Сложение (+) - сложение HEX чисел
- Вычитание (−) - вычитание HEX чисел
- Умножение (×) - умножение HEX чисел
- Деление (÷) - деление HEX чисел
- Остаток (%) - операция modulo для HEX чисел
- Возведение в степень(^) - Возведение для HEX чисел
- Поле для заметок с авто сохранением при изминении

Division (HEX) - Деление с остатком - получение частного и остатка от деления

Carrying out a number Mod n и Mod p - вычисление по модулю SECP256k1

HEX | DEC Converter - взаимное преобразование HEX и десятичных чисел

Bitcoin Address Generator:
- Legacy адреса (P2PKH) - начинаются с `1`
- SegWit адреса (P2SH-P2WPKH) - начинаются с `3`
- Native SegWit (P2WPKH) - начинаются с `bc1q`
- SegWit P2WSH - начинаются с `bc1`
- Taproot - начинаются с `bc1p`
- Детальная информация - публичные ключи, хэши, контрольные суммы

Division on Curve (EC Division) - Деление приватного ключа по правилам кривой Secp256k1

Search Plus Keys (Range Reduction) - Усечение приватного ключа на 1,2,3 и 4 бита.

Path Search Algorithm - Посмотреть 2 пути разложения ключа к привату 0x1: Верный и
Рандомный (Информативная версия)

(Верный путь - то как в в теории работал бы конечный алгоритм получения ключа.
В данномслучае - ключ раскладывается, делясь на 2 каждый раз,
добираясь до привата 0x1. Так же будет работать алгоритм, только с точками, когда будет
найдена уязвимость кривой;
Рандомный путь - то, как работает усечение на 1 бит, когда не знаешь
Какой из 2-х вариантов выбрать.)

Общая информация о кривой Secp256k1

Работает локально (127.0.0.1) на ПК пользователя.
Данные на сторонние сервера не передаются.
